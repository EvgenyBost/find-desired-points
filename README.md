# Line Intersections Clusterizer 📍

Консольное приложение на **C++/Qt**, предназначенное для поиска и анализа наиболее значимых точек пересечения множества прямых. Программа эффективно находит «узлы», где пересекается наибольшее количество линий, используя алгоритмы пространственной кластеризации.

## 🚀 Принцип работы

Алгоритм разбит на 6 ключевых этапов:
1.  **Чтение данных:** Загрузка $M$ прямых из текстового файла.
2.  **Поиск пересечений:** Математический расчет точек пересечения всех пар прямых внутри окна $[-1000, 1000]$.
3.  **Grid-кластеризация:** Группировка точек в кластеры с порогом расстояния $\le 2.0$ с использованием пространственного хеширования (ячейки сетки), что значительно быстрее простого перебора.
4.  **Сортировка (Ranking):** Кластеры сортируются по количеству содержащихся в них точек (от самых «плотных» к менее значимым).
5.  **Вычисление центроидов:** Для топ-$N$ кластеров рассчитывается центр масс.
6.  **Экспорт:** Результаты сортируются по координатам и сохраняются в файл.

## 🛠 Особенности реализации

-   **Оптимизация памяти:** Использование `reserve()` и `squeeze()` для минимизации переаллокаций векторов.
-   **Производительность:** Применение `QHash` с кастомным ключом (координаты сетки) позволяет обрабатывать сотни тысяч точек пересечения за секунды.
-   **Точность:** Центроиды вычисляются как среднее арифметическое всех координат внутри кластера для снижения погрешности.

## 📦 Установка и сборка

**Требования:**
- Qt 5.15+ или Qt 6.x
- Компилятор с поддержкой C++17

**Сборка:**
```bash
# Клонирование репозитория
git clone https://github.com
cd название-проекта

# Сборка через qmake
qmake
make
```

## 📥 Входные и выходные данные

### Входной файл (`check-lines.txt`)
Файл должен находиться в директории с программой:
```text
1000 5          # M (кол-во линий), N (кол-во искомых точек)
0 0 10 10       # Линия 1 (x1 y1 x2 y2)
...
```

### Выходной файл (`desired-points.txt`)
Список координат найденных центроидов:
```text
12.34 56.78
90.12 -34.56
```

## ⏱ Производительность
Программа выводит подробный лог выполнения в консоль:
- Количество найденных пересечений.
- Прогресс кластеризации в процентах.
- Общее время работы (Total operation time).
